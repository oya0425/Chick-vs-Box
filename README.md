# ひよこ無双：BOX壊滅 (Chick vs Box)

DirectX11 / C++ で制作した、スピード感あふれる3Dアクションゲームです。
逃げ回る「BoxUnity（敵）」を撃破し、全5WAVEのクリアを目指します。

## 🎮 ゲーム概要
時間経過で減少するHPを、敵を倒してコンボを繋ぐことで回復させる「リスク＆リターン」の要素を取り入れたWave制アクション。

- **全5WAVE構成**: WAVE進行に伴い、必要撃破数と難易度が上昇。
- **コンボシステム**: 連続撃破でHPを効率よく回復。
- **スピード感**: プレイヤーと敵の速度が段階的に上昇し、操作の緊張感を演出。

## 🛠 技術仕様
2週間という短期間で、ゲームエンジンを使わずDirectX11/C++を用いて基盤から実装しました。

- **開発環境**: Windows 10/11, Visual Studio 2022
- **グラフィックス**: DirectX 11 (Shader Model 5.0)
- **言語**: C++
- **実装内容**:
    - **オブジェクトプール**: 敵キャラクターの頻繁な生成・破棄による負荷を軽減。
    - **ゲームループ**: 更新と描画を分離した設計による安定した動作。
    - **UIアニメーション**: Tween（補間）による動的な拡縮・遷移演出。
    - **衝突判定**: AABBや球体判定を用いた3D空間の当たり判定。

## 🎨 こだわりポイント
- **Blenderによる自作モデル**: ひよこの3Dモデル作成から待機アニメーションまで構築。
- **直感的なフィードバック**: ボタンホバー時の拡縮・SEによる心地よい操作感。
- **動的なシーン遷移**: 回転と拡大を組み合わせた躍動感のある画面転換。
- **視認性を追求した色彩設計**: 背景とのコントラストを意識し、微細な色差を設けたUIデザイン。

## 🎮 操作方法
- **移動**: `W` `A` `S` `D`
- **カメラ**: マウス移動
- **ズーム**: マウスホイール

## ⬇️ プレイ方法
[Releases ページ](https://github.com/oya0425/Chick-vs-Box/releases)から最新のZIPファイルをダウンロードし、`ChickVsBox.exe` を実行してください。
